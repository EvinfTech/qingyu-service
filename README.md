## 1.项目简介

奇羽项目是一款智能软件，提供羽毛球预约、场地预约、后台用户管理、场地管理等多种功能，为用户提供便捷高效的体验。

### 1.1功能概况

1、奇羽项目使用golang gin、gorm技术，提高工作效率和创造力。<br/>
2、奇羽后台项目能够结合小程序和pc端进行场馆管理<br/>
